# A-star-Pathfinding
A quick demo of the A star search algorithm in Python.

# Usage
`.> astar.py`
  - First left click to set start point
  - Second left click to set end point
  - Click and drag to set walls
  - Press ENTER to run
  - Press ENTER again to reset once done
